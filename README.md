# Chess Game Review Website

A full-featured chess game analysis tool with Stockfish engine, interactive board, PGN import, and voice assistant "Magnus".

## Quick Start
```bash
npm install
npm run dev
```

Open http://localhost:3000

## Features
- PGN import (paste/upload/API)
- Interactive chessboard
- Stockfish 16 analysis
- Voice narration
- Game sharing
